Add a Location in Apple Business Connect

By adding a location, customers can see informations such as photos, highlights of new items, sales.
When you add the first brand, you can set logo and cover photo for all the locations under the the brand. When you add a new location, you can use a different cover photo for each location.

Add a new and unclaimed location

This guide will show you how to add the location Welpit under the brand Welpit that was added before.

  1. Select "Locations" in the sidebar, then select "Add" on the top-right.
  2. Enter your location name, then select "Add a new location with this name".
  3. Enter the location details about your business.
  4. Enter additional details if needed.
  5. Add the working hours of the location.
  6. Choose the brand, and select "Done".
  7. You will now be redirected to "Locations" tab, when your location is verified, it will be published across all Apple services. This process usually takes 4 days for an actual location.

Claim an existing location

  1. Follow the tutorial of "Adding a new location".
  2. When prompted to the brand page, you will have the option to add a new brand if need.
  3. Review the informations if needed.
  4. Verify your ownership of the location.
    • Verify by phone call
      1. Select "Location Verification", select "Verify by phone call", add optional extention if needed.
      2. Enter the 4-digit verification code.
      3. Your location will be immediately verified.
    • Verify by official document
      1. Select "Location Verification", select "Verify by official document".
      2. Upload the official document, such as utility bill or invoice, and add a short description to it.
      3. Upload the file to Apple, this process usually takes longer.
