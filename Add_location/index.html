How to Add a Location in Apple Business Connect

By adding a location, customers can see informations such as photos, highlights of new items, sales.
When you add the first brand, you can set logo and cover photo for all the locations under the the brand. When you add a new location, you can use a different cover photo for each location.

Add a new location

This tutorial will guide you through add a location that is new and not claimed.

  1. Select "Locations" in the sidebar, then select "Add" on the top-right.
  2. Enter your location name, then select "Add a new location with this name".
  3. Enter the location details about your business.
  4. Enter additional details if needed.
  5. Add the working hours of the location.
  6. Choose the brand, and select "Done".
  7. You will now be redirected to "Locations" tab, when your location is verified, it will be published across all Apple services. This process usually takes 4 days for an actual location.
